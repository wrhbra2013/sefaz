1. Hardware

  • Processador (CPU): Cérebro do computador
  • Memória RAM: Memória temporária
  • Disco rígido (HD/SSD): Armazenamento permanente
  • Placa-mãe: Conexão dos componentes

2. Software

TipoDescrição
Sistema operacionalWindows, Linux, MacOS
AplicativosWord, Excel, navegadores
DriversControladores de hardware

3. Redes de Computadores

  • LAN: Rede local
  • WAN: Rede ampla
  • Internet: Rede global
  • TCP/IP: Protocolo de comunicação

4. Banco de Dados

  • Modelo relacional: Tabelas com relações
  • SQL: Linguagem de consulta
  • CRUD: Create, Read, Update, Delete

5. Sistemas Operacionais

  • Windows: Microsoft
  • Linux: Código aberto (Ubuntu, CentOS)
  • macOS: Apple

6. Tipos de Dados

  • Estruturados: Bancos relacionais
  • Semiestruturados: JSON, XML
  • Não estruturados: Arquivos, imagens
