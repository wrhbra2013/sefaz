Resumo: A Matemática Financeira é fundamental para cálculos de juros, descontos e sistemas de amortização. Pratique bastante com exercícios!

1. Juros Simples

Fórmulas Fundamentais

J = C × i × n
  • J: Juros
  • C: Capital (principal)
  • i: Taxa de juros (em decimal)
  • n: Tempo (períodos)
M = C × (1 + i × n)
  • M: Montante (capital + juros)

2. Juros Compostos

Fórmulas Fundamentais

M = C × (1 + i)^n
J = M - C
  • Importante: No juros compostos, os juros incidem sobre o capital + juros anteriores (capitalização)

3. Taxas Equivalentes

(1 + ia) = (1 + ip)^12 = (1 + id)^360
  • ia: Taxa anual
  • ip: Taxa mensal
  • id: Taxa diária

4. Descontos

4.1 Desconto Simples (Comercial/Racional)

D = N × i × n
  • D: Desconto
  • N: Valor nominal (valor futuro)
  • i: Taxa de desconto
  • n: Tempo

4.2 Desconto Composto

V = N / (1 + i)^n
  • V: Valor atual (líquido)
  • N: Valor nominal

5. Sistemas de Amortização

SistemaCaracterísticas
SAF (Price)Parcelas fixas, juros decrescentes, amortização crescente
SACAmortização fixa, parcelas decrescentes, juros decrescentes
SAMMédia entre Price e SAC

6. Equivalência de Capitais

VF = VP × (1 + i)^n
  • VF: Valor futuro
  • VP: Valor presente
  • Regra: Dois capitais são equivalentes se, levados para uma mesma data (data focal), tiverem valores iguais

7. Taxa Interna de Retorno (TIR)

  • Taxa que zera o VPL (Valor Presente Líquido)
  • Critério de decisão: aceitar se TIR > taxa de atratividade
  • VPN = 0: Σ Fluxos/(1+TIR)^t = 0
