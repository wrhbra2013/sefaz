1. Conceitos Fundamentais

  • Confidencialidade: Apenas autorizados acesso
  • Integridade: Dados não alterados
  • Disponibilidade: Acesso quando necessário
  • Autenticidade: Verificar identidade

2. Criptografia

TipoUso
SimétricaMesma chave (AES, DES)
AssimétricaChaves pública/privada (RSA)
HashIntegridade (SHA, MD5)

3. Firewall

  • Filtragem de pacotes: Controlar tráfego
  • Stateful inspection: Monitorar conexões
  • Proxy: Intermediateira

4. Ataques e Vulnerabilidades

  • Malware: Vírus, ransomware, spyware
  • Phishing: Golpe de engenharia social
  • DDoS: Negação de serviço
  • SQL Injection: Ataque a banco de dados

5. Backup

  • Incremental: Apenas alterações
  • Completo: Todos os dados
  • Diferencial: Mudanças desde último completo
  • Regra 3-2-1: 3 cópias, 2 mídias, 1 off-site

6. ISO 27001/27002

  • 27001: Sistemas de gestão
  • 27002: Controles de segurança
  • PDCA: Plan, Do, Check, Act
Dica: Estude os pilares CIA e ISO 27001. São fundamentais para segurança.
